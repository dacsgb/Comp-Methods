scipy.special.itj0y0¶

scipy.special.itj0y0(x) = <ufunc 'itj0y0'>¶

Integrals of Bessel functions of order 0

Returns simple integrals from 0 to x of the zeroth order Bessel functions j0 and y0.

Returns
ij0, iy0
